The application is calculate the Basal Metabolic rate using height, weight, age, sex.
The autolayout is used to make application responsive. There are two unit 
(1) Metric
(2) US Customary

The application build on MVC architecture.

The user need to input height, weight, age, sex and selects the unit and then application calculate the BMR.
 
There is also 1 more feature included if user need to loose 5 pounds in 3 months how much calories he/she needs in daily.

The singleton Instance is used in model.

The description of every files is also included in beginning of the files.
